# Radar Ventas ML Bot

Bot automático para el canal @radarventasml en Telegram.
Postea ofertas de Mercado Libre Argentina con links de afiliado.

## Variables de entorno necesarias en Railway:
- TELEGRAM_TOKEN → token del bot (de BotFather)
- AFFILIATE_ID → tu ID de afiliado de Mercado Libre

## El bot hace:
1. Busca productos con descuento en ML Argentina
2. Genera link con tu ID de afiliado
3. Postea al canal cada 4 horas automático
